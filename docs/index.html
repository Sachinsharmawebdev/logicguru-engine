Logic Guru Engine

A powerful, async-ready JSON-based logic rule engine for evaluating nested conditions, variable bindings, and custom actions.

Install from NPM View on GitHub

Key Features

Nested Conditions

  • Logical: and, or
  • Comparison: ==, !=, >, <, >=, <=
  • Array/Object: includeIn, includeKey, includeVal

Date Functions

  • year(date): Calculate years
  • month(date): Calculate months
  • day(date): Calculate days

Dynamic Actions

  • log: Template-based logging
  • assign: Create new objects
  • update: Modify nested paths
  • excludeVal: Remove array values

Quick Start

Installation

npm install logicguru-engine

Basic Usage

import { configureRuleEngine } from "logicguru-engine";

const rules = [
  {
    "id": "age-verification",
    "condition": {
      "and": [
        { ">=": ["${year($context.birthDate)}", 18] }
      ]
    },
    "actions": [
      {
        "type": "assign",
        "key": "result.isAdult",
        "value": true
      }
    ]
  }
];

const context = {
  birthDate: "2000-01-01"
};

const engine = await configureRuleEngine(rules, {
  basePath: "./data",
  defaultContext: context
});

const result = await engine();
console.log(result);

Advanced Features

Context Variables

Use $ variables resolved from provided context with template string support:

{
  "condition": {
    "and": [
      { "==": ["$user.status", "active"] },
      { ">=": ["${year($user.joinDate)}", 1] }
    ]
  }
}

Dynamic File Loading

Load external data with template path support:

{
  "useFiles": {
    "configFile": {
      "path": "/config/${$context.type}_config.json"
    }
  }
}
